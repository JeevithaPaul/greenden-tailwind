# 🌿 Greenden

**Greenden** is a responsive website designed using **HTML** and **Tailwind CSS**. It offers a clean, modern user interface that introduces students to real-world website layout practices. The project focuses on using Tailwind’s utility-first classes to build attractive, flexible, and responsive designs.

---

## 🚀 Features

- Fully responsive design (mobile, tablet, desktop)
- Built using Tailwind CSS utility-first approach
- Clean and modern UI
- Beginner-friendly layout structure
- Lightweight and fast-loading pages

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
